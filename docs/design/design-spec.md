# 교시표 디자인 기준

## 기준 시안

- `desktop-concept.png`: 1536×1024 데스크톱 화면, 우측 수업 추가 패널 상태
- `mobile-concept.png`: 853×1844 모바일 화면, 하단 수업 추가 시트 상태
- 두 PNG는 구현 참고용이며 런타임 UI 자산으로 사용하지 않는다.

## 허용된 첫 화면 문구

- 교시표
- 초기화
- 수업 추가
- 월요일, 화요일, 수요일, 목요일, 금요일
- 1교시~8교시
- 이 기기에 자동 저장됨

## 시각 시스템

- 배경: 순백색 `#ffffff`
- 본문 글자: 차콜 `#171717`
- 보조 글자: 중간 회색 `#666a73`
- 격자: 차가운 연회색 `#e7e8eb`
- 기본 버튼: 거의 검정 `#111111`
- 과목 블록: 라벤더, 블루, 민트, 옐로, 오렌지, 핑크, 그린, 그레이의 저채도 단색
- 모서리: 입력 8px, 수업 블록 6px, 모바일 시트 상단 20px
- 그림자와 그라디언트는 사용하지 않는다.

## 반응형 구조

- 1024px 이상: 시간표와 폭 424px의 우측 편집 패널을 나란히 배치한다.
- 1023px 이하: 편집기는 화면 아래에서 올라오는 고정 시트가 된다.
- 480px 이하: 월~금 전체와 교시 축을 한 화면 폭에 유지한다. 수업 보조 정보는 한 줄 말줄임 처리한다.

## 아이콘

- 추가: 24×24, 1.75px 선의 직선형 플러스
- 닫기: 24×24, 1.75px 선의 대각선 X
- 삭제: 20×20, 1.75px 선의 휴지통
- 모두 `currentColor`, round cap/join을 사용한다.
